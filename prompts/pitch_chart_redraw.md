# Pitching biomechanics chart prompt

将输入的投球事件和指标绘制为论文风格的独立 PNG 图表。

- 使用白色背景、清晰坐标轴和可打印配色。
- 不同单位使用双 Y 轴，禁止把角度和速度放在同一数值轴上。
- 标注抬腿最高点、前脚落地和出手事件；事件标签不得遮挡曲线。
- 每条曲线的峰值用圆点、细引线、时间和数值标注。
- 图例、标题、副标题、峰值标签不得重叠。
- 若曲线由事件锚点重建而不是逐帧原始序列，图注必须写明“presentation curve reconstructed from event metrics”，不得称为原始时序。
- 只输出图表，不添加装饰插画或运动员肖像。
