# Bundled report-card action illustrations

These source images are the reusable, unannotated action illustrations copied
from the Julian reference report. They are intentionally tracked because a
fresh report output directory must be able to render its report-card assets
without depending on a prior generated report.

- `batting/frontend_metric_illustrations/`: 14 source PNGs used by
  `annotate_frontend_metric_illustrations.py`.
- `pitching/lineart_actions/`: three source PNGs used by
  `annotate_pitch_lineart_metrics.py`; that script writes the player-specific
  `*_metrics.png` variants during each pitching execution.
